**Seven Cats Inn** is a tavern located on the southeastern edges of [Novigrad](https://witcher.fandom.com/wiki/Novigrad "Novigrad"), just near the bridge to the [Southern Gate](https://witcher.fandom.com/wiki/Southern_Gate "Southern Gate"). 

The name is fitting as one can clearly see there are seven different [cats](https://witcher.fandom.com/wiki/Cat_\(creature\) "Cat \(creature\)") roaming the area. 

## Map description

    _This dank establishment is host to a shady clientele._
