## Appearance(s)

### Games

_[The Witcher 3: Wild Hunt](https://witcher.fandom.com/wiki/The_Witcher_3:_Wild_Hunt "The Witcher 3: Wild Hunt")_

The **Druids' Camp** is a small encampment located in the western forests of [Ard Skellig](https://witcher.fandom.com/wiki/Ard_Skellig "Ard Skellig"). There is an [herbalist](https://witcher.fandom.com/wiki/Herbalist_\(Druids%27_Camp\) "Herbalist \(Druids' Camp\)") that can be found here. 

## Map description

    _Camp pitched by[druids](https://witcher.fandom.com/wiki/Druid "Druid") investigating the magic cataclysm which devastated the nearby woods._
