# Elite Crossbow set

[![Tw3 dlc crossbow set](https://static.wikia.nocookie.net/witcher/images/3/3c/Tw3_dlc_crossbow_set.jpg/revision/latest/scale-to-width-down/360?cb=20170601224800)](https://static.wikia.nocookie.net/witcher/images/3/3c/Tw3_dlc_crossbow_set.jpg/revision/latest?cb=20170601224800) [](https://witcher.fandom.com/wiki/File:Tw3_dlc_crossbow_set.jpg)

The **Elite Crossbow Set** is one of the 16 [free DLCs](https://witcher.fandom.com/wiki/Category:The_Witcher_3_DLC "Category:The Witcher 3 DLC") released for _[The Witcher 3: Wild Hunt](https://witcher.fandom.com/wiki/The_Witcher_3:_Wild_Hunt "The Witcher 3: Wild Hunt")_ on June 3, 2015. The DLC includes three new [crossbows](https://witcher.fandom.com/wiki/Crossbow "Crossbow"). 

All three are of master quality and share most base attributes, but each is sold only by a chosen [merchant](https://witcher.fandom.com/wiki/The_Witcher_3_merchants "The Witcher 3 merchants"). The table below includes only differing details.
