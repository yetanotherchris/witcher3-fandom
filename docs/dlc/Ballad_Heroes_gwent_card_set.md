# Ballad Heroes gwent card set

[![Tw3 dlc ballad heroes](https://static.wikia.nocookie.net/witcher/images/3/3a/Tw3_dlc_ballad_heroes.jpg/revision/latest/scale-to-width-down/349?cb=20170601224315)](https://static.wikia.nocookie.net/witcher/images/3/3a/Tw3_dlc_ballad_heroes.jpg/revision/latest?cb=20170601224315) [](https://witcher.fandom.com/wiki/File:Tw3_dlc_ballad_heroes.jpg)

**Ballad Heroes** is one of the 16 [free DLCs](https://witcher.fandom.com/wiki/Category:The_Witcher_3_DLC "Category:The Witcher 3 DLC") for _[The Witcher 3: Wild Hunt](https://witcher.fandom.com/wiki/The_Witcher_3:_Wild_Hunt "The Witcher 3: Wild Hunt")_ that brings a new appearance to several [gwent](https://witcher.fandom.com/wiki/Gwent "Gwent") cards. It was released on 12 June 2015. 

The card set can be enabled in the Downloadable Content tab of the main menu. 

## Official Statement

_Make your Gwent Deck legendary! The "Ballad Heroes" neutral Gwent card set will provide alternative cards capturing characters from[Dandelion](https://witcher.fandom.com/wiki/Dandelion "Dandelion")'s ballads in all their glory._
