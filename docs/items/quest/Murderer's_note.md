## Item ID

q308_anneke_invite

**Murderer's note** is a [quest item](https://witcher.fandom.com/wiki/The_Witcher_3_quest_items "The Witcher 3 quest items") in _[The Witcher 3: Wild Hunt](https://witcher.fandom.com/wiki/The_Witcher_3:_Wild_Hunt "The Witcher 3: Wild Hunt")_. 

This is the note found by [Geralt](https://witcher.fandom.com/wiki/Geralt_of_Rivia "Geralt of Rivia") on [Patricia Vegelbud](https://witcher.fandom.com/wiki/Patricia_Vegelbud "Patricia Vegelbud")'s corpse during the quest [Carnal Sins](https://witcher.fandom.com/wiki/Carnal_Sins "Carnal Sins"). It exists in the game files and can be obtained through console commands, but whether due to a bug or intentionally, doesn't appear during gameplay.
