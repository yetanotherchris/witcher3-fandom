## Item ID

q111_falkas_coin

**Defier's Oren** is a [quest item](https://witcher.fandom.com/wiki/The_Witcher_3_quest_items "The Witcher 3 quest items") in _[The Witcher 3: Wild Hunt](https://witcher.fandom.com/wiki/The_Witcher_3:_Wild_Hunt "The Witcher 3: Wild Hunt")_. 

You must retrieve this [coin](https://witcher.fandom.com/wiki/Oren "Oren") from an underwater cavern after [Thecla](https://witcher.fandom.com/wiki/Thecla "Thecla") drops it there as a test during the [main quest](https://witcher.fandom.com/wiki/The_Witcher_3_main_quests "The Witcher 3 main quests"), [Bald Mountain](https://witcher.fandom.com/wiki/Bald_Mountain_\(quest\) "Bald Mountain \(quest\)").
