## Item ID

q401_sausages

**Sausages** are a [cut content](https://witcher.fandom.com/wiki/The_Witcher_3_cut_content "The Witcher 3 cut content") [quest item](https://witcher.fandom.com/wiki/The_Witcher_3_quest_items "The Witcher 3 quest items") in _[The Witcher 3: Wild Hunt](https://witcher.fandom.com/wiki/The_Witcher_3:_Wild_Hunt "The Witcher 3: Wild Hunt")_ , still present in the game files, but only obtainable through console commands. It was somehow associated with the main quest [Ugly Baby](https://witcher.fandom.com/wiki/Ugly_Baby "Ugly Baby"), which by virtue of being an "umbrella" quest makes two possible purposes plausible. 

The first is a scene during the [No Place Like Home](https://witcher.fandom.com/wiki/No_Place_Like_Home "No Place Like Home") quest, where [Geralt](https://witcher.fandom.com/wiki/Geralt_of_Rivia "Geralt of Rivia") was supposed to raid the kitchen and bring some food back to the table - something that is now only mentioned in dialogue right before [Eskel](https://witcher.fandom.com/wiki/Eskel "Eskel") disappears, having gone for more vodka because Geralt already went for food earlier (and brought [bread](https://witcher.fandom.com/wiki/Bread_\(The_Witcher_3\)#Quest_item "Bread \(The Witcher 3\)") and [cheese](https://witcher.fandom.com/wiki/Cheese_\(The_Witcher_3\)#Quest_item "Cheese \(The Witcher 3\)"), which can be seen on the table). This is a more plausible option because of the items order of appearance in the game files, as well as the item description. 

The second option is that it was supposed to be a special ingredient for making [omelet à la Geralt of Rivia](https://witcher.fandom.com/wiki/Omelet_%C3%A0_la_Geralt_of_Rivia "Omelet à la Geralt of Rivia") during a cut cooking phase of [Va Fail, Elaine](https://witcher.fandom.com/wiki/Va_Fail,_Elaine "Va Fail, Elaine"). Unfortunately, this doesn't actually work there, even though that part of the quest can be re-enabled via console commands or mods. 

Neither can be discounted since the item is listed to appear in two loot containers, a "vodka cabinet" and an "ingredients cabinet".
