## Item ID

mq1022_paint

**Paint** is a [quest item](https://witcher.fandom.com/wiki/The_Witcher_3_quest_items "The Witcher 3 quest items") in _[The Witcher 3: Wild Hunt](https://witcher.fandom.com/wiki/The_Witcher_3:_Wild_Hunt "The Witcher 3: Wild Hunt")_. If the secondary quest, [The Volunteer](https://witcher.fandom.com/wiki/The_Volunteer "The Volunteer"), is done and one decides to help [Trollololo](https://witcher.fandom.com/wiki/Trollololo "Trollololo"), he'll request you to fetch him some paint so he can paint the Redanian coat of arms. 

This item can be purchased from the [merchant](https://witcher.fandom.com/wiki/Merchant_\(Oxenfurt\) "Merchant \(Oxenfurt\)") in [Oxenfurt](https://witcher.fandom.com/wiki/Oxenfurt "Oxenfurt") (and can be purchased before starting the quest).
