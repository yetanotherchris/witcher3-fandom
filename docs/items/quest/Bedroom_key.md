## Item ID

q001_bedroom_key

**Bedroom key** is a [quest item](https://witcher.fandom.com/wiki/The_Witcher_3_quest_items "The Witcher 3 quest items") and key in _[The Witcher 3: Wild Hunt](https://witcher.fandom.com/wiki/The_Witcher_3:_Wild_Hunt "The Witcher 3: Wild Hunt")_. 

It is the first item that can be looted in the game and is needed to unlock the bedroom doors at the very beginning of the first main quest, [Kaer Morhen](https://witcher.fandom.com/wiki/Kaer_Morhen_\(quest\) "Kaer Morhen \(quest\)"). As that quest serves as a tutorial, during which the [inventory](https://witcher.fandom.com/wiki/The_Witcher_3_inventory "The Witcher 3 inventory") isn't enabled yet, the bedroom key cannot be observed as an inventory item unless added via console commands later.
