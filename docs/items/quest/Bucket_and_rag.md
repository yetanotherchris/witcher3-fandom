## Item ID

q401_bucket_and_rag

**Bucket and rag** is a [cut content](https://witcher.fandom.com/wiki/The_Witcher_3_cut_content "The Witcher 3 cut content") [quest item](https://witcher.fandom.com/wiki/The_Witcher_3_quest_items "The Witcher 3 quest items") in _[The Witcher 3: Wild Hunt](https://witcher.fandom.com/wiki/The_Witcher_3:_Wild_Hunt "The Witcher 3: Wild Hunt")_ , still present in the game files, but only obtainable through console commands. 

During the main quest [Va Fail, Elaine](https://witcher.fandom.com/wiki/Va_Fail,_Elaine "Va Fail, Elaine"), this was supposed to be manually retrieved when [Uma](https://witcher.fandom.com/wiki/Uma "Uma") vomits, before the cutscene where [Geralt](https://witcher.fandom.com/wiki/Geralt_of_Rivia "Geralt of Rivia") and [Eskel](https://witcher.fandom.com/wiki/Eskel "Eskel") are cleaning up the mess.
