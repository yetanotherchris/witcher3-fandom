## Item ID

sq201_chamber_key

**Temple key** is a [quest item](https://witcher.fandom.com/wiki/The_Witcher_3_quest_items "The Witcher 3 quest items") in _[The Witcher 3: Wild Hunt](https://witcher.fandom.com/wiki/The_Witcher_3:_Wild_Hunt "The Witcher 3: Wild Hunt")_. 

It exists in the game files and can be obtained through console commands, but doesn't seem to appear during gameplay and its purpose is unknown, although it is somehow related to [In Wolf's Clothing](https://witcher.fandom.com/wiki/In_Wolf%27s_Clothing "In Wolf's Clothing"). 

The first obvious thing to try with it is the front gate to [Freya's Garden](https://witcher.fandom.com/wiki/Hindar "Hindar"), but it doesn't unlock it. A more likely candidate are the big doors at [Freya's temple](https://witcher.fandom.com/wiki/Temple_of_Freyja "Temple of Freyja") to the east (where [Josta](https://witcher.fandom.com/wiki/Josta "Josta") is), which however have been disabled from interacting with (i.e. there is a prompt, but there's no response like the usual _"Locked. Key required."_ message).
