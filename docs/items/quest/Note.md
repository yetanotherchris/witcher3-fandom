## Item ID

poi_bandit_camp_3_note

This **Note** is part of [cut content](https://witcher.fandom.com/wiki/The_Witcher_3_cut_content "The Witcher 3 cut content") in [White Orchard](https://witcher.fandom.com/wiki/White_Orchard "White Orchard"). 

While its appearance is that of a [notice board posting](https://witcher.fandom.com/wiki/Category:The_Witcher_3_notice_board_postings "Category:The Witcher 3 notice board postings"), data in the game files suggests that it was supposed to be found in the [bandit camp](https://witcher.fandom.com/wiki/Bandit_camp "Bandit camp") northwest of [Cackler Bridge](https://witcher.fandom.com/wiki/Cackler_Bridge "Cackler Bridge"), or is otherwise related to it.
