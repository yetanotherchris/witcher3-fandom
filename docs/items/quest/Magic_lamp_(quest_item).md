## Item ID

q106_magic_oillamp

**Magic Lamp** is a [quest item](https://witcher.fandom.com/wiki/The_Witcher_3_quest_items "The Witcher 3 quest items") in _[The Witcher 3: Wild Hunt](https://witcher.fandom.com/wiki/The_Witcher_3:_Wild_Hunt "The Witcher 3: Wild Hunt")_. 

It's initially picked up by [Keira Metz](https://witcher.fandom.com/wiki/Keira_Metz "Keira Metz") during [Magic Lamp](https://witcher.fandom.com/wiki/Magic_Lamp_\(quest\) "Magic Lamp \(quest\)"), but she'll give it to [Geralt](https://witcher.fandom.com/wiki/Geralt_of_Rivia "Geralt of Rivia") later if one chooses to do [An Invitation from Keira Metz](https://witcher.fandom.com/wiki/An_Invitation_from_Keira_Metz "An Invitation from Keira Metz"). 

The item can be used anywhere spirits and such ghosts are found (typically indicated by green lights). 

The lamp can be used as a light source too, but unlike a [torch](https://witcher.fandom.com/wiki/Torch "Torch") it only emits light in the front and it prevents Geralt from running, however unlike a torch it will not ignite swamp gas making it a safer option when traversing through it.
