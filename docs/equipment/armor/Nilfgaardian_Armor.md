## Item ID

DLC5 Nilfgaardian Armor (regular game)  
NGP DLC5 Nilfgaardian Armor ([New Game +](https://witcher.fandom.com/wiki/New_Game_%2B "New Game +"))

**Nilfgaardian Armor** is part of the [Nilfgaardian Armor set](https://witcher.fandom.com/wiki/Nilfgaardian_Armor_set "Nilfgaardian Armor set") [free DLC](https://witcher.fandom.com/wiki/Category:The_Witcher_3_DLC "Category:The Witcher 3 DLC") released for _[The Witcher 3: Wild Hunt](https://witcher.fandom.com/wiki/The_Witcher_3:_Wild_Hunt "The Witcher 3: Wild Hunt")_ on 3 June 2015. It can be bought from the [quartermaster](https://witcher.fandom.com/wiki/Quartermaster_\(Crow%27s_Perch\) "Quartermaster \(Crow's Perch\)") in [Crow's Perch](https://witcher.fandom.com/wiki/Crow%27s_Perch "Crow's Perch") and cannot be dismantled. 

Whether due to a bug or intentionally, the upgraded [New Game +](https://witcher.fandom.com/wiki/New_Game_%2B "New Game +") version can be purchased in a regular game as well. At level 35, it will replace the level 10 version in the quartermaster's inventory. Similarly, it cannot be purchased before reaching level 35 even in a New Game +.
