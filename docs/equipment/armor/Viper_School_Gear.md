# Viper School Gear

[![Viper medallion](https://static.wikia.nocookie.net/witcher/images/e/e3/Tw2_trailer_schooloftheviper.png/revision/latest/scale-to-width-down/100?cb=20160416054726)](https://static.wikia.nocookie.net/witcher/images/e/e3/Tw2_trailer_schooloftheviper.png/revision/latest?cb=20160416054726) [](https://witcher.fandom.com/wiki/File:Tw2_trailer_schooloftheviper.png)

Viper medallion

**Viper School Gear** is a set of [armor](https://witcher.fandom.com/wiki/Armor "Armor") and [swords](https://witcher.fandom.com/wiki/Swords "Swords") from _[The Witcher 3: Wild Hunt](https://witcher.fandom.com/wiki/The_Witcher_3:_Wild_Hunt "The Witcher 3: Wild Hunt")_ and its _[Hearts of Stone](https://witcher.fandom.com/wiki/Hearts_of_Stone "Hearts of Stone")_ expansion. The set was designed for [witchers](https://witcher.fandom.com/wiki/Witcher "Witcher") that were part of the [School of the Viper](https://witcher.fandom.com/wiki/Viper_School "Viper School"). 

_Note:_ all legendary gear are stronger versions of base game gear in [New Game +](https://witcher.fandom.com/wiki/New_Game_%2B "New Game +"). 

## _The Witcher 3: Wild Hunt_

In the base game the set only includes two low leveled swords, one silver and one steel, previously owned by [Kolgrim](https://witcher.fandom.com/wiki/Kolgrim "Kolgrim") from the School of the Viper. When you collect the first weapon schematic, it will start a new [Scavenger Hunt](https://witcher.fandom.com/wiki/Scavenger_Hunt:_Viper_School_Gear "Scavenger Hunt: Viper School Gear") secondary quest. 

  * [Viper silver sword](https://witcher.fandom.com/wiki/Viper_silver_sword "Viper silver sword")
  * [Viper steel sword](https://witcher.fandom.com/wiki/Viper_steel_sword "Viper steel sword")

## _Hearts of Stone_ expansion

The full armor set is introduced in the _Hearts of Stone_ expansion. There is no secondary quest associated with this gear, meaning that it can be completely missed as they can only be obtained during certain quests: 

  * The armor diagrams are bought from [Countess Mignole](https://witcher.fandom.com/wiki/Mignole "Mignole") during the [Open Sesame!](https://witcher.fandom.com/wiki/Open_Sesame!_\(Hearts_of_Stone\) "Open Sesame! \(Hearts of Stone\)") quest
  * The venomous steel sword diagram is found within the Borsodi vault during the [Open Sesame!](https://witcher.fandom.com/wiki/Open_Sesame!_\(Hearts_of_Stone\) "Open Sesame! \(Hearts of Stone\)") quest
  * The venomous silver sword has no crafting diagram, but the forged blade can be found within [Gaunter O'Dimm](https://witcher.fandom.com/wiki/Gaunter_O%27Dimm "Gaunter O'Dimm")'s [world](https://witcher.fandom.com/wiki/Gaunter%27s_world "Gaunter's world")

For a more detailed walkthrough on getting all the pieces, see [Scavenger Hunt: Viper School Gear](https://witcher.fandom.com/wiki/Scavenger_Hunt:_Viper_School_Gear "Scavenger Hunt: Viper School Gear").
