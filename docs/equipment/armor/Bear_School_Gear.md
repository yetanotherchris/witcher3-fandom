# Bear School Gear

[![Full mastercrafted set](https://static.wikia.nocookie.net/witcher/images/a/a4/Bear.png/revision/latest/scale-to-width-down/321?cb=20151009153209)](https://static.wikia.nocookie.net/witcher/images/a/a4/Bear.png/revision/latest?cb=20151009153209) [![Full mastercrafted set](https://static.wikia.nocookie.net/witcher/images/a/a4/Bear.png/revision/latest/scale-to-width-down/321?cb=20151009153209)](https://static.wikia.nocookie.net/witcher/images/a/a4/Bear.png/revision/latest?cb=20151009153209) [](https://witcher.fandom.com/wiki/File:Bear.png)

Full mastercrafted set

**Bear** or **Ursine School Gear** is a [witcher](https://witcher.fandom.com/wiki/Witcher "Witcher") armor and swords set inspired by the [Bear School](https://witcher.fandom.com/wiki/Bear_School "Bear School") for [witchers](https://witcher.fandom.com/wiki/Witcher "Witcher") and is also one of the [Scavenger Hunts](https://witcher.fandom.com/wiki/Scavenger_Hunt:_Bear_School_Gear "Scavenger Hunt: Bear School Gear") quests in _[The Witcher 3: Wild Hunt](https://witcher.fandom.com/wiki/The_Witcher_3:_Wild_Hunt "The Witcher 3: Wild Hunt")_. The set includes chest armor, steel and silver swords, a crossbow, gauntlets, boots, and trousers, all of which are upgradeable except the crossbow. Also, there is not a [New Game +](https://witcher.fandom.com/wiki/New_Game_%2B "New Game +") version of the crossbow. 

The gear is some of the best [heavy armor](https://witcher.fandom.com/wiki/The_Witcher_3_heavy_armor "The Witcher 3 heavy armor") found in _[The Witcher 3: Wild Hunt](https://witcher.fandom.com/wiki/The_Witcher_3:_Wild_Hunt "The Witcher 3: Wild Hunt")_. The initial set can be found in [Skellige](https://witcher.fandom.com/wiki/Skellige "Skellige"), at the peak of [Yngvar's Fang](https://witcher.fandom.com/wiki/Yngvar%27s_Fang "Yngvar's Fang") located north of [Urialla Harbor](https://witcher.fandom.com/wiki/Urialla_Harbor "Urialla Harbor"). 

  * **Bonuses for 3 pieces:** When a [Quen](https://witcher.fandom.com/wiki/Quen "Quen") shield shatters, there is a chance a new one will be cast at no Stamina cost. The bonus increases by 5% for each piece of the set.
  * **Bonuses for 6 pieces:** Damage dealt by [Abilities](https://witcher.fandom.com/wiki/The_Witcher_3_abilities "The Witcher 3 abilities") involving the Quen [Sign](https://witcher.fandom.com/wiki/Sign "Sign") is increased by 200%.

_Note:_ all legendary gear are stronger versions of base game gear in New Game +.
