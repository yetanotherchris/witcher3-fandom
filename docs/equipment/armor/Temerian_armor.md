**Temerian armor** returns in the third game as [light](https://witcher.fandom.com/wiki/The_Witcher_3_light_armor "The Witcher 3 light armor") [armor](https://witcher.fandom.com/wiki/The_Witcher_3_chest_armor "The Witcher 3 chest armor") and is available with the [Temerian Armor set](https://witcher.fandom.com/wiki/Temerian_Armor_set "Temerian Armor set") [free DLC](https://witcher.fandom.com/wiki/Category:The_Witcher_3_DLC "Category:The Witcher 3 DLC"). The entire set can be bought from [Bram](https://witcher.fandom.com/wiki/Bram "Bram") in [White Orchard](https://witcher.fandom.com/wiki/White_Orchard "White Orchard"). 

Whether due to a bug or intentionally, the upgraded [New Game +](https://witcher.fandom.com/wiki/New_Game_%2B "New Game +") version can be purchased in a regular game as well. At level 29, it will replace the level 4 version in Bram's inventory. Similarly, it cannot be purchased before reaching level 29 even in a New Game +.
