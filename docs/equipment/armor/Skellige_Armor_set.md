# Skellige Armor set

[![Tw3 dlc skellige armor set](https://static.wikia.nocookie.net/witcher/images/f/f9/Tw3_dlc_skellige_armor_set.jpg/revision/latest/scale-to-width-down/300?cb=20170601224924)](https://static.wikia.nocookie.net/witcher/images/f/f9/Tw3_dlc_skellige_armor_set.jpg/revision/latest?cb=20170601224924) [![Tw3 dlc skellige armor set](https://static.wikia.nocookie.net/witcher/images/f/f9/Tw3_dlc_skellige_armor_set.jpg/revision/latest/scale-to-width-down/300?cb=20170601224924)](https://static.wikia.nocookie.net/witcher/images/f/f9/Tw3_dlc_skellige_armor_set.jpg/revision/latest?cb=20170601224924) [](https://witcher.fandom.com/wiki/File:Tw3_dlc_skellige_armor_set.jpg)

Enhanced for the frigid climate of the [Skellige Archipelago](https://witcher.fandom.com/wiki/Skellige "Skellige"), this set includes a jacket, gloves, pants, boots and horse tack. It is one of the 16 [free DLCs](https://witcher.fandom.com/wiki/Category:The_Witcher_3_DLC "Category:The Witcher 3 DLC") for _[The Witcher 3: Wild Hunt](https://witcher.fandom.com/wiki/The_Witcher_3:_Wild_Hunt "The Witcher 3: Wild Hunt")_ on June 24 2015. Although the DLC calls it [Skellige](https://witcher.fandom.com/wiki/Skellige "Skellige") armor, the in-game items are referred to as [Undvik](https://witcher.fandom.com/wiki/Undvik "Undvik") armor. It can be bought from the [Armorer](https://witcher.fandom.com/wiki/Armorer_\(Kaer_Trolde_keep\) "Armorer \(Kaer Trolde keep\)") in [Kaer Trolde citadel](https://witcher.fandom.com/wiki/Kaer_Trolde_citadel "Kaer Trolde citadel"). The most basic version of the armor set is equippable at level 16 or above, while the second version becomes available at level 41.
