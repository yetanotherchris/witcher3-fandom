# Temerian Armor set

[![Tw3 dlc temerian armor set](https://static.wikia.nocookie.net/witcher/images/a/a4/Tw3_dlc_temerian_armor_set.jpg/revision/latest/scale-to-width-down/300?cb=20170601225020)](https://static.wikia.nocookie.net/witcher/images/a/a4/Tw3_dlc_temerian_armor_set.jpg/revision/latest?cb=20170601225020) [![Tw3 dlc temerian armor set](https://static.wikia.nocookie.net/witcher/images/a/a4/Tw3_dlc_temerian_armor_set.jpg/revision/latest/scale-to-width-down/300?cb=20170601225020)](https://static.wikia.nocookie.net/witcher/images/a/a4/Tw3_dlc_temerian_armor_set.jpg/revision/latest?cb=20170601225020) [](https://witcher.fandom.com/wiki/File:Tw3_dlc_temerian_armor_set.jpg)

Reserved for the finest of warriors serving the kingdom of [Temeria](https://witcher.fandom.com/wiki/Temeria "Temeria"), the **Temerian Armor Set** includes a jacket, gloves, pants, boots and horse tack. It is one of the 16 [free DLCs](https://witcher.fandom.com/wiki/Category:The_Witcher_3_DLC "Category:The Witcher 3 DLC") for _[The Witcher 3: Wild Hunt](https://witcher.fandom.com/wiki/The_Witcher_3:_Wild_Hunt "The Witcher 3: Wild Hunt")_ that was released on June 3, 2015. The entire set can be bought from [Bram](https://witcher.fandom.com/wiki/Bram "Bram") in [White Orchard](https://witcher.fandom.com/wiki/White_Orchard "White Orchard"). The most basic version of the armor set is equippable at level 4 or above, while the second version becomes available at level 29.
