# Spikeroog aketon

**Spikeroog aketon** is a [light](https://witcher.fandom.com/wiki/The_Witcher_3_light_armor "The Witcher 3 light armor") [armor](https://witcher.fandom.com/wiki/The_Witcher_3_chest_armor "The Witcher 3 chest armor") in _[The Witcher 3: Wild Hunt](https://witcher.fandom.com/wiki/The_Witcher_3:_Wild_Hunt "The Witcher 3: Wild Hunt")_. It has the following versions: 

  * Spikeroog aketon (relic)
  * Spikeroog aketon (crafted)
  * Spikeroog aketon (basic)
