## Item ID

Mutagen 3

**Cockatrice decoction** is a [potion](https://witcher.fandom.com/wiki/The_Witcher_3_potions "The Witcher 3 potions") in _[The Witcher 3: Wild Hunt](https://witcher.fandom.com/wiki/The_Witcher_3:_Wild_Hunt "The Witcher 3: Wild Hunt")_ that adds 1 extra charge to all craftable [bombs](https://witcher.fandom.com/wiki/The_Witcher_3_bombs "The Witcher 3 bombs") and standard potions. Despite what the description says however, it will not grant extra charges to mutagen decoctions.

## Bugs

  * The decoction will not work when Geralt has additional charges from [Corvo Bianco](https://witcher.fandom.com/wiki/Corvo_Bianco "Corvo Bianco")'s alchemy lab, thus drinking it will do absolutely nothing. 
    * However, it will work with the additional charges from [Manticore School Gear](https://witcher.fandom.com/wiki/Manticore_School_Gear "Manticore School Gear").
    * The bug can be bypassed by using any of the alchemy creations first. Meaning if the creation lost at least one of it's charges it will gain the additional charge when this potion is consumed.
