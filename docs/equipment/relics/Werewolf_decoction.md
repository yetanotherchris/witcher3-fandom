**Werewolf decoction** is a [potion](https://witcher.fandom.com/wiki/The_Witcher_3_potions "The Witcher 3 potions") in _[The Witcher 3: Wild Hunt](https://witcher.fandom.com/wiki/The_Witcher_3:_Wild_Hunt "The Witcher 3: Wild Hunt")_. 

Since the "next-gen" [Patch 4.0](https://witcher.fandom.com/wiki/Patch_4.0_\(The_Witcher_3\) "Patch 4.0 \(The Witcher 3\)"), it increases stamina regeneration in combat by 50% when fighting on clear weather nights.  
Previously, it made running and jumping outside of combat cost no stamina. However, this became redundant as patch 4.0 made that to be the default behavior.

## Tips

  * The decoction works between 6pm - 7am, as long as the weather is clear.
  * The description is misleading as the regeneration is affected by [Geralt](https://witcher.fandom.com/wiki/Geralt_of_Rivia "Geralt of Rivia")'s armor weight class and will only be at default 50 when wearing [medium armor](https://witcher.fandom.com/wiki/The_Witcher_3_medium_armor "The Witcher 3 medium armor"). 
    * When wearing [heavy armor](https://witcher.fandom.com/wiki/The_Witcher_3_heavy_armor "The Witcher 3 heavy armor"), the regeneration will be 37.5
    * When wearing [light armor](https://witcher.fandom.com/wiki/The_Witcher_3_light_armor "The Witcher 3 light armor"), the regeneration will be 62.5
  * Using [glyphwords](https://witcher.fandom.com/wiki/Enchanting#Glyphwords "Enchanting") that change armor weight class will change the regeneration accordingly. Thus wearing [Levity](https://witcher.fandom.com/wiki/Levity "Levity")-enchanted gear will grant the full 62.5 regeneration.
