## Attributes

### Effect(s)

Releases a cloud of gas that exploded when ignited. Enhanced effect and chance to apply Burning. Foes it kills explode, wounding others nearby.  
30s Effect duration  
**Regular game:**  
300 Fire damage (400 before [Patch 4.0](https://witcher.fandom.com/wiki/Patch_4.0_\(The_Witcher_3\) "Patch 4.0 \(The Witcher 3\)"))  
**[New Game +](https://witcher.fandom.com/wiki/New_Game_%2B "New Game +"):**  
450-600 Fire damage (600 before [Patch 4.0](https://witcher.fandom.com/wiki/Patch_4.0_\(The_Witcher_3\) "Patch 4.0 \(The Witcher 3\)"))

## Item ID

Dragons Dream 3

**Superior Dragon's Dream** is a [bomb](https://witcher.fandom.com/wiki/The_Witcher_3_bombs "The Witcher 3 bombs") in _[The Witcher 3: Wild Hunt](https://witcher.fandom.com/wiki/The_Witcher_3:_Wild_Hunt "The Witcher 3: Wild Hunt")_ that releases a cloud of gas that explodes when ignited. Without any [special abilities](https://witcher.fandom.com/wiki/The_Witcher_3_abilities "The Witcher 3 abilities"), up to four of these bombs can be carried at one time.
