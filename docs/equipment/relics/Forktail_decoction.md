## Tips

  * The description is not written properly, in order to gain the boost one must use 3 different ways of attack - counter, sword attack, sign, [bomb](https://witcher.fandom.com/wiki/The_Witcher_3_bombs "The Witcher 3 bombs") or [crossbow](https://witcher.fandom.com/wiki/Crossbow "Crossbow"). Using any combination of 3 among them, except fast and strong attacks as both count as a sword attack, will grant the boost. 
    * It doesn't seem to matter if the attacks actually hit or not, with the exception of counters of course.
    * 3 different attacks have to be performed in the span of 4 seconds. "Fact" of each attack type being done expires after 4 seconds. Bonus is granted if upon attack player has 3 different "attack facts". It is possible to immediately regain bonus, after bonus is used.
  * The boost will reset after landing the next strike on an enemy.
  * The boost will not reset when using [signs](https://witcher.fandom.com/wiki/Sign "Sign") except [Axii](https://witcher.fandom.com/wiki/Axii "Axii"), so it is possible to keep it as long as other signs are used and no sword attacks are performed. (this is most likely a bug) 
    * However, alternate sign modes will reset the boost with the exception of [Yrden](https://witcher.fandom.com/wiki/Yrden "Yrden").
  * As with most stat boosting effects it only takes base damage into account.
