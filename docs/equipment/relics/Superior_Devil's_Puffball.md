## Bugs

  * The crafting screen will incorrectly display 30s effect duration instead of the 33 properly shown on the actual bomb. This is caused by how the item is defined in the game files, with 30s listed directly on the item and 3s added by its effect definition, the latter of which is not read by the crafting screen.
