## Bugs

  * The crafting screen will incorrectly display 10s effect duration instead of the 13, properly shown on the actual bomb. This is caused by how the item is defined in the game files, with 10s listed directly on the item and 3s added by its effect definition, the latter of which is not read by the crafting screen.
