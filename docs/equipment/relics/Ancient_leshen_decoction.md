**Ancient leshen decoction** is a [potion](https://witcher.fandom.com/wiki/The_Witcher_3_potions "The Witcher 3 potions") in _[The Witcher 3: Wild Hunt](https://witcher.fandom.com/wiki/The_Witcher_3:_Wild_Hunt "The Witcher 3: Wild Hunt")_ that increases stamina regeneration in combat each time a [Sign](https://witcher.fandom.com/wiki/Sign "Sign") is cast. The bonus lasts until combat ends, and can stack without limit. 

_Note:_ Prior to [Patch 4.0](https://witcher.fandom.com/wiki/Patch_4.0_\(The_Witcher_3\) "Patch 4.0 \(The Witcher 3\)"), this potion was bugged and its effects only triggered with alternate sign modes (except [Yrden](https://witcher.fandom.com/wiki/Yrden "Yrden")), and reportedly provided 5 stamina regeneration instead of 2.

## Tips

  * Although [Patch 4.0](https://witcher.fandom.com/wiki/Patch_4.0_\(The_Witcher_3\) "Patch 4.0 \(The Witcher 3\)") fixes the decoction making alternate sign mode non-mandatory it is still much faster than using signs normally.
  * The decoction works very well with [Quen](https://witcher.fandom.com/wiki/Quen "Quen")'s alternative mode, [Active Shield](https://witcher.fandom.com/wiki/Active_Shield "Active Shield"), as when at maximum it does not use up stamina without getting hit by an enemy. Thus it is possible to spam the alternate sign mode to build up regeneration.
  * [Aard](https://witcher.fandom.com/wiki/Aard "Aard") Sweep works, however it uses up stamina thus cannot be spammed so much at the beginning of a fight.
  * [Igni](https://witcher.fandom.com/wiki/Igni "Igni")'s alternate mode, Firestream, is a poor choice as the regeneration does not increase if [Geralt](https://witcher.fandom.com/wiki/Geralt_of_Rivia "Geralt of Rivia") gets interrupted while using it.
  * The effect works very well with the [ekhidna decoction](https://witcher.fandom.com/wiki/Ekhidna_decoction "Ekhidna decoction"), especially when combined with Active Shield. Once the regeneration builds up Geralt will become practically unkillable.
  * The regeneration is affected by Geralt's armor weight class and will only be at default 2 when wearing medium gear. 
    * When wearing heavy gear the regeneration will be 1.5
    * When wearing light gear the regeneration will be 2.5
