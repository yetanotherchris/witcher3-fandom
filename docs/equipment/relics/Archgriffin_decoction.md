## Tip

  * The potion can be a huge pro or con depending on how you use it. Since when fighting enemies against which regular attacks already deal more than 5% damage, the potion is then pretty much useless and will only keep using up your stamina. Thus it should only be used during fights againsts bosses that have massive health bars, in which case it will prevent the need to butter knife the enemy to death.
  * Very useful in the _[Hearts of Stone](https://witcher.fandom.com/wiki/Hearts_of_Stone "Hearts of Stone")_ expansion as boss enemies from it tend to be very durable, most notably [The Toad Prince](https://witcher.fandom.com/wiki/The_Toad_Prince "The Toad Prince") and [Olgierd von Everec](https://witcher.fandom.com/wiki/Olgierd_von_Everec "Olgierd von Everec").
