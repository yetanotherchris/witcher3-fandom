# Gwent players

Throughout _[The Witcher 3: Wild Hunt](https://witcher.fandom.com/wiki/The_Witcher_3:_Wild_Hunt "The Witcher 3: Wild Hunt")_ , there are a number of [gwent](https://witcher.fandom.com/wiki/Gwent "Gwent") players. This is a complete list of every player, separated by game and regions. 

**†** This denotes players that can die or disappear from the game. With [Patch 1.08](https://witcher.fandom.com/wiki/Patch_1.08_\(The_Witcher_3\) "Patch 1.08 \(The Witcher 3\)"), these cards can be found at locations, typically near where the character resided.
