# Hybrid

**Hybrids** are a type of [monster](https://witcher.fandom.com/wiki/The_Witcher_3_bestiary "The Witcher 3 bestiary") in _[The Witcher 3: Wild Hunt](https://witcher.fandom.com/wiki/The_Witcher_3:_Wild_Hunt "The Witcher 3: Wild Hunt")_. Hybrids include: 

  * [Archgriffin](https://witcher.fandom.com/wiki/Archgriffin "Archgriffin")
  * [Ekhidna](https://witcher.fandom.com/wiki/Ekhidna "Ekhidna")
  * [Erynia](https://witcher.fandom.com/wiki/Erynia "Erynia")
  * [Griffin](https://witcher.fandom.com/wiki/Griffin_\(creature\) "Griffin \(creature\)")
  * [Harpy](https://witcher.fandom.com/wiki/Harpy "Harpy")
  * [Melusine](https://witcher.fandom.com/wiki/Melusine "Melusine")
  * [Opinicus](https://witcher.fandom.com/wiki/Opinicus "Opinicus")
  * [Salma](https://witcher.fandom.com/wiki/Salma "Salma")
  * [Siren](https://witcher.fandom.com/wiki/Siren "Siren")
  * [Succubus](https://witcher.fandom.com/wiki/Succubus "Succubus")
