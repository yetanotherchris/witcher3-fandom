**Fourth letter from Reinald** is a document in _[The Witcher 3: Wild Hunt](https://witcher.fandom.com/wiki/The_Witcher_3:_Wild_Hunt "The Witcher 3: Wild Hunt")_ that was introduced with the next-gen update, [Patch 4.0](https://witcher.fandom.com/wiki/Patch_4.0_\(The_Witcher_3\) "Patch 4.0 \(The Witcher 3\)"). 

It can be found in the [Devil's Pit](https://witcher.fandom.com/wiki/Devil%27s_Pit "Devil's Pit") mine system during [In the Eternal Fire's Shadow](https://witcher.fandom.com/wiki/In_the_Eternal_Fire%27s_Shadow "In the Eternal Fire's Shadow"), in a small dead end while following the [School of the Wolf](https://witcher.fandom.com/wiki/School_of_the_Wolf "School of the Wolf") symbols.
