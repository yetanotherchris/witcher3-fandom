## Item ID

lw_gr15_bandits_note

This **orders** version is a [cut content](https://witcher.fandom.com/wiki/The_Witcher_3_cut_content "The Witcher 3 cut content") book in _[The Witcher 3: Wild Hunt](https://witcher.fandom.com/wiki/The_Witcher_3:_Wild_Hunt "The Witcher 3: Wild Hunt")_ , still present in the game files, but only obtainable through console commands. 

It was supposed to be found on a [Nilfgaardian soldier](https://witcher.fandom.com/wiki/Nilfgaardian_Army "Nilfgaardian Army")'s corpse at either the [Ferry Station](https://witcher.fandom.com/wiki/Ferry_Station "Ferry Station") or [Calveit Watchtower](https://witcher.fandom.com/wiki/Calveit_Watchtower "Calveit Watchtower"), before the area is liberated. There are corpses in both places using the exact loot container template that is supposed to have this, but whether due to a bug or oversight only the template's appearance is used.
