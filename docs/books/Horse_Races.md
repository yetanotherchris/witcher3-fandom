## Item ID

hr101_notice

**Horse Races** is a notice board posting in _[The Witcher 3: Wild Hunt](https://witcher.fandom.com/wiki/The_Witcher_3:_Wild_Hunt "The Witcher 3: Wild Hunt")_. 

Reading it is one way to start [Races: Crow's Perch](https://witcher.fandom.com/wiki/Races:_Crow%27s_Perch "Races: Crow's Perch"). 

## Journal entry

    _If you haven't heard, we[race](https://witcher.fandom.com/wiki/Horse_racing "Horse racing") [horses](https://witcher.fandom.com/wiki/Horse "Horse") around here. Anyone wants to join, come talk to me, I'll fill you in on how it works. But I'll tell you right now, you gotta pay an entry fee to race, so if you're skint, fuck off._
