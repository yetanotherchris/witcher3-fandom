## Item ID

Meteorite ore

**Meteorite ore** is a [crafting component](https://witcher.fandom.com/wiki/The_Witcher_3_crafting_components "The Witcher 3 crafting components") in _[The Witcher 3: Wild Hunt](https://witcher.fandom.com/wiki/The_Witcher_3:_Wild_Hunt "The Witcher 3: Wild Hunt")_ that is needed to craft several items. 

It can be found as random loot or purchased from the following [merchants](https://witcher.fandom.com/wiki/The_Witcher_3_merchants "The Witcher 3 merchants"):
