## Item ID

Resin

**Resin** is a [crafting component](https://witcher.fandom.com/wiki/The_Witcher_3_crafting_components "The Witcher 3 crafting components") in _[The Witcher 3: Wild Hunt](https://witcher.fandom.com/wiki/The_Witcher_3:_Wild_Hunt "The Witcher 3: Wild Hunt")_ that can be dismantled into [sap](https://witcher.fandom.com/wiki/Sap "Sap"). It is needed to craft the following items: 

  * [Hardened timber](https://witcher.fandom.com/wiki/Hardened_timber "Hardened timber")
  * [Ursine crossbow](https://witcher.fandom.com/wiki/Ursine_crossbow "Ursine crossbow")
