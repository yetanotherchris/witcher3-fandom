## Item ID

Thread

**Thread** is a [crafting component](https://witcher.fandom.com/wiki/The_Witcher_3_crafting_components "The Witcher 3 crafting components") in _[The Witcher 3: Wild Hunt](https://witcher.fandom.com/wiki/The_Witcher_3:_Wild_Hunt "The Witcher 3: Wild Hunt")_ that can be dismantled into [cotton](https://witcher.fandom.com/wiki/Cotton "Cotton"). It is needed to craft many items. 

**Crafting Requirements** ![Tw2 icon crafting](https://static.wikia.nocookie.net/witcher/images/9/9e/Tw2_icon_crafting.svg/revision/latest?cb=20110531194409)  

**Min. Required Level:** Amateur [armorer](https://witcher.fandom.com/wiki/Armorer "Armorer")/[blacksmith](https://witcher.fandom.com/wiki/Blacksmith "Blacksmith")
