## Item ID

**Twine** is a [crafting component](https://witcher.fandom.com/wiki/The_Witcher_3_crafting_components "The Witcher 3 crafting components") that can be dismantled into [fiber](https://witcher.fandom.com/wiki/Fiber "Fiber").
