## Item ID

Timber

**Timber** is a [crafting component](https://witcher.fandom.com/wiki/The_Witcher_3_crafting_components "The Witcher 3 crafting components") that can be dismantled into [ashes](https://witcher.fandom.com/wiki/Ashes "Ashes"). It is needed to craft the following items: 

  * [Blunt crossbow bolt](https://witcher.fandom.com/wiki/Blunt_crossbow_bolt "Blunt crossbow bolt")
  * [Bolt with bait](https://witcher.fandom.com/wiki/Bolt_with_bait "Bolt with bait")
  * [Broadhead bolt](https://witcher.fandom.com/wiki/Broadhead_bolt "Broadhead bolt")
  * [Exploding bolt](https://witcher.fandom.com/wiki/Exploding_bolt "Exploding bolt")
  * [Hardened timber](https://witcher.fandom.com/wiki/Hardened_timber "Hardened timber")
  * [Precision bolt](https://witcher.fandom.com/wiki/Precision_bolt "Precision bolt")
  * [Split bolt](https://witcher.fandom.com/wiki/Split_bolt "Split bolt")
  * [Tracker's trousers](https://witcher.fandom.com/wiki/Tracker%27s_trousers "Tracker's trousers")
  * [Tracking bolt](https://witcher.fandom.com/wiki/Tracking_bolt "Tracking bolt")

It can be found as regular random loot, random drops from non-ancient [leshens](https://witcher.fandom.com/wiki/Leshy "Leshy"), or purchased from the following [merchants](https://witcher.fandom.com/wiki/The_Witcher_3_merchants "The Witcher 3 merchants"):
