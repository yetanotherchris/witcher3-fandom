## Item ID

Empty bottle

**Empty bottle** is an [alchemy ingredient](https://witcher.fandom.com/wiki/The_Witcher_3_alchemy_ingredients "The Witcher 3 alchemy ingredients") in _[The Witcher 3: Wild Hunt](https://witcher.fandom.com/wiki/The_Witcher_3:_Wild_Hunt "The Witcher 3: Wild Hunt")_ that is needed to craft the following items: 

  * [Alcohest](https://witcher.fandom.com/wiki/Alcohest "Alcohest")
  * [Dwarven spirit](https://witcher.fandom.com/wiki/Dwarven_spirit "Dwarven spirit")
  * [White Gull](https://witcher.fandom.com/wiki/White_Seagull "White Seagull")
