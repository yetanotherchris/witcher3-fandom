## Item ID

Noonwraith light essence

**Light essence** is an [alchemy ingredient](https://witcher.fandom.com/wiki/The_Witcher_3_alchemy_ingredients "The Witcher 3 alchemy ingredients") in _[The Witcher 3: Wild Hunt](https://witcher.fandom.com/wiki/The_Witcher_3:_Wild_Hunt "The Witcher 3: Wild Hunt")_ that can be dismantled into [monster essence](https://witcher.fandom.com/wiki/Monster_essence "Monster essence"). It is needed to craft the following items: 

  * [Golden Oriole](https://witcher.fandom.com/wiki/Golden_Oriole "Golden Oriole")
  * [Enhanced Golden Oriole](https://witcher.fandom.com/wiki/Enhanced_Golden_Oriole "Enhanced Golden Oriole")

It is a guaranteed drop from all [noonwraiths](https://witcher.fandom.com/wiki/Noonwraith "Noonwraith").
