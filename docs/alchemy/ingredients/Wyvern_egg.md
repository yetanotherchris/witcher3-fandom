## Item ID

Wyvern egg

**Wyvern egg** is an [alchemy ingredient](https://witcher.fandom.com/wiki/The_Witcher_3_alchemy_ingredients "The Witcher 3 alchemy ingredients"). It is not needed to craft anything but can be dismantled into [monster egg](https://witcher.fandom.com/wiki/Monster_egg "Monster egg"), [monster claw](https://witcher.fandom.com/wiki/Monster_claw "Monster claw"), and [chitinous shell](https://witcher.fandom.com/wiki/Chitinous_shell "Chitinous shell"). 

It is a guaranteed drop from all [wyverns](https://witcher.fandom.com/wiki/Wyvern "Wyvern").
