## Item ID

Basilisk venom

**Basilisk venom** is an [alchemy ingredient](https://witcher.fandom.com/wiki/The_Witcher_3_alchemy_ingredients "The Witcher 3 alchemy ingredients") in _[The Witcher 3: Wild Hunt](https://witcher.fandom.com/wiki/The_Witcher_3:_Wild_Hunt "The Witcher 3: Wild Hunt")_. It is not needed for anything but can be dismantled into [venom extract](https://witcher.fandom.com/wiki/Venom_extract "Venom extract"). 

It is a guaranteed drop from all [basilisks](https://witcher.fandom.com/wiki/Basilisk "Basilisk").
