## Item ID

Optima mater

**Optima mater** is an [alchemy ingredient](https://witcher.fandom.com/wiki/The_Witcher_3_alchemy_ingredients "The Witcher 3 alchemy ingredients") that is needed to craft the following items: 

  * [Dimeritium bomb](https://witcher.fandom.com/wiki/Dimeritium_bomb "Dimeritium bomb")
  * [Enhanced Dimeritium bomb](https://witcher.fandom.com/wiki/Enhanced_Dimeritium_bomb "Enhanced Dimeritium bomb")
  * [Superior dimeritium bomb](https://witcher.fandom.com/wiki/Superior_dimeritium_bomb "Superior dimeritium bomb")
  * [Enhanced Dragon's Dream](https://witcher.fandom.com/wiki/Enhanced_Dragon%27s_Dream "Enhanced Dragon's Dream")
  * [Superior Dragon's Dream](https://witcher.fandom.com/wiki/Superior_Dragon%27s_Dream "Superior Dragon's Dream")
  * [Mastercrafted Ursine armor](https://witcher.fandom.com/wiki/Mastercrafted_Ursine_armor "Mastercrafted Ursine armor")
  * [Mastercrafted legendary Ursine armor](https://witcher.fandom.com/wiki/Mastercrafted_legendary_Ursine_armor "Mastercrafted legendary Ursine armor")

It can be found as random loot or purchased from the following [merchants](https://witcher.fandom.com/wiki/The_Witcher_3_merchants "The Witcher 3 merchants"): 

  * All [alchemists](https://witcher.fandom.com/wiki/Alchemist "Alchemist") except the [Eternal Fire Priest](https://witcher.fandom.com/wiki/Eternal_Fire_Priest_\(Electors%27_Square\) "Eternal Fire Priest \(Electors' Square\)")
  * All [herbalists](https://witcher.fandom.com/wiki/Herbalist "Herbalist") outside of [Toussaint](https://witcher.fandom.com/wiki/Toussaint "Toussaint"), except [Yolar](https://witcher.fandom.com/wiki/Yolar "Yolar")
