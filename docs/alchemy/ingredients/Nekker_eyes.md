## Item ID

Nekker eye

**Nekker eye** is an [alchemy ingredient](https://witcher.fandom.com/wiki/The_Witcher_3_alchemy_ingredients "The Witcher 3 alchemy ingredients") that can be dismantled into [monster eye](https://witcher.fandom.com/wiki/Monster_eye "Monster eye"). It is needed to craft [Enhanced Hanged Man's Venom](https://witcher.fandom.com/wiki/Enhanced_Hanged_Man%27s_Venom "Enhanced Hanged Man's Venom"). 

It is a random drop from [nekkers](https://witcher.fandom.com/wiki/Nekker "Nekker") and their [monster nests](https://witcher.fandom.com/wiki/Monster_nest "Monster nest").
