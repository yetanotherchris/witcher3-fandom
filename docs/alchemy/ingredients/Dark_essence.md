## Item ID

Nightwraith dark essence

**Dark essence** is an [alchemy ingredient](https://witcher.fandom.com/wiki/The_Witcher_3_alchemy_ingredients "The Witcher 3 alchemy ingredients") in _[The Witcher 3: Wild Hunt](https://witcher.fandom.com/wiki/The_Witcher_3:_Wild_Hunt "The Witcher 3: Wild Hunt")_ that can be dismantled into [monster essence](https://witcher.fandom.com/wiki/Monster_essence "Monster essence"). It is needed to craft the following items: 

  * [Full Moon](https://witcher.fandom.com/wiki/Full_Moon "Full Moon")
  * [Enhanced Full Moon](https://witcher.fandom.com/wiki/Enhanced_Full_Moon "Enhanced Full Moon")

It is a guaranteed drop from all [nightwraiths](https://witcher.fandom.com/wiki/Nightwraith "Nightwraith").
