**Gargoyle dust** is used to craft the following items: 

  * [Kinslayer's boots](https://witcher.fandom.com/wiki/Kinslayer%27s_boots "Kinslayer's boots")
  * [Kinslayer's gauntlets](https://witcher.fandom.com/wiki/Kinslayer%27s_gauntlets "Kinslayer's gauntlets")
  * [Myrmidon's gauntlets](https://witcher.fandom.com/wiki/Myrmidon%27s_gauntlets "Myrmidon's gauntlets")
  * [Mourner](https://witcher.fandom.com/wiki/Mourner "Mourner")

### Notes

If you wish to craft the [Kinslayer's Outfit](https://witcher.fandom.com/wiki/Dark_Mode "Dark Mode") in [Dark Mode](https://witcher.fandom.com/wiki/Dark_Mode "Dark Mode"), make sure to farm all 9 dusts before deactivating the final ward for [The Gargoyle Contract](https://witcher.fandom.com/wiki/The_Gargoyle_Contract "The Gargoyle Contract"). 

Nearby gargoyles will no longer respawn once the ward for the area is deactivated. Gargoyle dust is not sold by any merchants and there are no other respawning gargoyles.

## Item ID

Gargoyle dust

**Gargoyle dust** is an [alchemy ingredient](https://witcher.fandom.com/wiki/The_Witcher_3_alchemy_ingredients "The Witcher 3 alchemy ingredients"). It is not needed to craft anything but can be dismantled into [powdered monster tissue](https://witcher.fandom.com/wiki/Powdered_monster_tissue "Powdered monster tissue"). 

It is a random drop from all [gargoyles](https://witcher.fandom.com/wiki/Gargoyle "Gargoyle").
