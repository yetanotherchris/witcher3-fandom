## Item ID

Forktail plate

**Forktail hide** is an [alchemy ingredient](https://witcher.fandom.com/wiki/The_Witcher_3_alchemy_ingredients "The Witcher 3 alchemy ingredients") in _[The Witcher 3: Wild Hunt](https://witcher.fandom.com/wiki/The_Witcher_3:_Wild_Hunt "The Witcher 3: Wild Hunt")_. It is not needed to craft anything but can be dismantled into a [monster carapace](https://witcher.fandom.com/wiki/Monster_carapace "Monster carapace") (the base game one) and [chitinous shell](https://witcher.fandom.com/wiki/Chitinous_shell "Chitinous shell"). 

It is a random drop from all [forktails](https://witcher.fandom.com/wiki/Forktail "Forktail").
