## Item ID

Gargoyle heart

**Gargoyle heart** is an [alchemy ingredient](https://witcher.fandom.com/wiki/The_Witcher_3_alchemy_ingredients "The Witcher 3 alchemy ingredients"). It is not needed to craft anything but can be dismantled into [monster heart](https://witcher.fandom.com/wiki/Monster_heart "Monster heart") and [powdered monster tissue](https://witcher.fandom.com/wiki/Powdered_monster_tissue "Powdered monster tissue"). 

It is a random drop from all [gargoyles](https://witcher.fandom.com/wiki/Gargoyle "Gargoyle").
