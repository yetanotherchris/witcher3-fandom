## Item ID

Maribor Forest 2

**Enhanced Maribor Forest** is a [potion](https://witcher.fandom.com/wiki/The_Witcher_3_potions "The Witcher 3 potions") in _[The Witcher 3: Wild Hunt](https://witcher.fandom.com/wiki/The_Witcher_3:_Wild_Hunt "The Witcher 3: Wild Hunt")_ , being an upgraded version of [Maribor Forest](https://witcher.fandom.com/wiki/Maribor_Forest_\(potion\) "Maribor Forest \(potion\)"). Four of these can be crafted at any one time and placed in the consumables slot. 

It gives exactly the same amount of Adrenaline Point generation as the basic version, but lasts twice as long.
